# 결과물 평가 (자기평가)

신주용 · SKALA 4기 · Full-stack Engineering (HTML / CSS / JavaScript)

> 모든 결과물은 프레임워크·빌드도구 없이 **순수 HTML / CSS / JavaScript**로 작성함.

---

## 요약

- 순수 HTML/CSS/JavaScript만으로 교안 7페이지 + 종합실습 4개 프로젝트 완성 (프레임워크·빌드도구 미사용).
- 각 과제의 필수·세부·추가 요구사항 전부 충족.
- 요구사항을 넘어, 교안 실습을 응용한 나만의 기능을 다수 직접 구현.

### 직접 구현한 추가 기능

| 과제 | 추가로 구현한 기능 |
|---|---|
| 교안 | 커스텀 유효성 메시지 · 비밀번호 보기·강도·글자수 · 날씨 자동순환 · 유튜브 지연 로딩 · 개발자 운세 |
| ① 회원가입 | 관심분야 12종 · 커뮤니티 가이드라인 필수 동의 · 제출값 표 렌더 · 비밀번호 마스킹 |
| ② 여행지 | 명소 이미지 라이트박스 · 가보고 싶은 곳 체크리스트 · 유튜브 4개 자동 순환 · 내비 섹션 하이라이트 |
| ③ 상품 갤러리 | 카테고리 필터 · 장바구니 · 찜하기 · 가격 정렬 (전부 `localStorage` 연동) |
| ④ 할 일 앱 | 마감일(지난 날짜 강조) · 우선순위 태그·정렬 · 진행률 바 · 인라인 편집 · 예시 데이터 |

> 추가 기능은 모두 소스 주석에 `[추가과제]` / `[보완]`으로 명시하고, 실행 캡처에도 박스로 강조 표시함.

---

## 1. 교안 과제

**구현 내용**
- HTML 7페이지: 나의 소개, 강의 시간표, 휴일 일과, 여행지, 회원가입/결과, 메인 Hub.
- CSS(`style.css`): 구글 폰트, 색·박스모델, 폼, Flexbox/Grid, 반응형 미디어쿼리, 애니메이션.
- JavaScript: 업다운 게임, 성적 계산기, 내 가방(객체 배열), 실시간 날씨(DOM/이벤트 → 비동기 fetch → 모듈 분리).

**배운 점**
- HTML 뼈대 → CSS 스타일 → JavaScript 동작의 점진적 확장 흐름을 체득함.
- 표의 `rowspan`/`colspan` 병합, Flex/Grid 반응형, `fetch`·`async/await`, `import/export` 모듈 분리에서 개념이 잡힘.

**잘한 점**
- 시맨틱 태그(`header/nav/main/section/article/aside/footer`)를 일관되게 사용함.
- 반응형(3열→1열)과 등장 애니메이션까지 적용함.
- 예제를 베끼지 않고 콘텐츠를 직접 채워 나만의 결과물로 만듦.

**아쉬운 점**
- 날씨 API가 외부 서비스라 오프라인에서는 확인 불가.
- 게임 결과를 `alert`으로 출력해 화면 UI와 분리됨(추후 DOM 출력으로 개선 여지).

**추가과제 (교안)**

| 추가과제 | 구현 / 의도 |
|---|---|
| 커스텀 유효성 메시지 | 브라우저 기본 경고 대신 직접 정한 안내 문구를 `setCustomValidity`로 표시 |
| 회원가입 폼 고도화 | 비밀번호 보기/숨기기, 강도 실시간 미터, 자기소개 글자수 카운터 |
| 날씨 자동 순환 | 토글 시 `setInterval`로 15초마다 도시 자동 전환 |
| 유튜브 지연 로딩 | 썸네일 표시 후 클릭 시 iframe 로드(facade 패턴)로 초기 로딩 비용 절감 |
| 오늘의 개발자 운세 | 배열 + `Math.random`으로 방문/버튼마다 랜덤 문구 표시 |

→ 기본 요구사항을 넘어 이벤트 처리·`localStorage`·비동기·지연 로딩을 응용함.

## 2. 종합실습

### ① 회원가입 & 결과 페이지
- **구현**: 모든 입력 유형(text/password/email/tel/date/radio/checkbox/select/textarea), 유효성 속성(required/pattern/minlength/min-max), `method=get` 쿼리스트링 전송, 결과 페이지에서 제출값을 표로 표시(복수 체크박스 포함), 고정 헤더(`position:fixed`).
- **추가**: 관심분야 12종, **커뮤니티 가이드라인 필수 동의**(미동의 시 가입 차단), 비밀번호 결과 표 마스킹.
- **배운 점**: 자바스크립트 없이 브라우저 기본 검증만으로 상당한 유효성 처리가 가능함을 확인함. `URLSearchParams`로 쿼리스트링을 파싱해 결과 표를 동적으로 생성함.

### ② 여행지 소개 페이지 (제주)
- **구현**: `header/nav/main/section/article/aside/footer` 시맨틱 구조, 고정 `nav` + 섹션 `id` 앵커 이동(`scroll-behavior: smooth`), 명소 `figure`+`figcaption` 6개, 먹거리 `article`+`aside` 2단, 히어로 gradient+이미지 오버레이, 카드 hover 확대, "↑ 맨 위로".
- **추가**: 명소 이미지 라이트박스(클릭 확대·배경/ESC/버튼 닫기), 가보고 싶은 명소 체크리스트(`localStorage`), 유튜브 4개 자동 순환(15초 전환 + 이전/다음 + 진행 바), 스크롤 시 현재 섹션 내비 하이라이트.
- **배운 점**: 시맨틱 구조와 앵커 내비게이션, 반응형 2단 레이아웃, 라이트박스·캐러셀 인터랙션을 라이브러리 없이 순수 JS로 구현하는 법을 익힘.

### ③ 반응형 상품 카드 갤러리
- **구현**: `:root` 색 변수 다수 + 전 구간 `var()`, Grid 3열(카드 6개), Flexbox 상단바, 반응형(900px 2열 / 600px 1열+메뉴 숨김), 카드 hover, 히어로 `@keyframes`, 다크모드(변수 교체 + `localStorage`), `prefers-reduced-motion` 대응, 실제 상품 이미지.
- **추가**: 카테고리 필터(메뉴 실동작), 장바구니 담기(카운터+localStorage), 상품 찜하기(카운터+localStorage), 가격 정렬(낮은/높은순).
- **배운 점**: CSS 변수로 다크모드를 값 교체만으로 구현하는 방식, 미디어쿼리 브레이크포인트 설계, 상태(찜/장바구니)를 `localStorage`로 영속화하는 패턴을 익힘.

### ④ 인터랙티브 할 일 관리 앱
- **구현**: 추가(Enter/버튼·빈 값 방지), 완료 토글(취소선), 삭제, 전체/진행중/완료 필터 + 요약, 이벤트 위임(`closest`), `storage.js`/`app.js` 모듈 분리(`type=module`), `localStorage` 저장·복원, "오늘의 한마디"(로컬 JSON fetch + 실패 시 기본 문구 폴백).
- **추가**: 마감일 입력·표시(지난 날짜 강조), 우선순위(색상 태그+정렬), 진행률 바(완료율 % 시각화), 항목 인라인 편집(더블클릭 → Enter), 예시 데이터 5종.
- **배운 점**: 데이터(`storage.js`)와 화면(`app.js`)을 모듈로 분리하는 구조, 이벤트 위임으로 동적 항목을 하나의 리스너로 처리하는 방식, 상태를 `localStorage`에 JSON으로 영속화하는 패턴을 익힘.

## 3. 총평

- **기본기 우선**: 프레임워크·빌드도구 없이 순수 HTML/CSS/JavaScript만으로 교안 7페이지 + 종합실습 4개를 완성함. 웹의 3요소(구조·표현·동작)를 직접 손으로 익히는 데 집중함.
- **요구사항 완수**: 각 과제의 필수·세부·추가 요구사항을 빠짐없이 충족하고, 실행 결과를 캡처로 남김.
- **응용과 확장**: 요구사항에 머무르지 않고 `localStorage`·비동기 `fetch`·모듈 분리·라이트박스·캐러셀·다크모드·진행률 바 같은 인터랙션을 직접 구현함.
- **다음 목표**: 접근성(키보드 내비, ARIA)과 코드 재사용(컴포넌트화)을 더 다듬고, 이 기본기를 바탕으로 프레임워크(예: Vue)로 확장할 계획임.

> 전체 소스와 실행 캡처는 각 과제 폴더에 정리되어 있으며, 추가 구현 기능은 소스 주석과 캡처 강조 표시로 확인 가능함.
